## Entry Thesis (2014)

ดร.นิเวศน์ เลือก TCAP เพราะตรง 4 เกณฑ์หลัก:

1. **PE 7-8 เท่า** — ถูกเทียบกับ banking sector
2. **PBV < 1.0** — ซื้อ Equity ของบริษัทที่ราคาต่ำกว่าทุน
3. **Dividend yield ~5%** — มี cash return ทันที
4. **Mid-tier bank → growth runway** — ไม่ใช่ big-4 ที่อิ่มตัว

> "ราคาหุ้นที่ถูกก็พอจะชดเชยประเด็นตรงนี้"
>
> "ความปลอดภัย และผลตอบแทนที่ดีกว่าดอกเบี้ยเงินฝากธนาคาร"
>

>

## Hidden Value: TMB Stake

ก่อนการรวมธนาคารธนชาตกับ TMB ในปี 2562–2563, TCAP ถือหุ้นใน Bank of Ayudhya (BAY) บางส่วน + ธนาคารธนชาตเอง — โครงสร้าง holding ที่หลายคนมอง ข้าม

หลังรวมธนชาต-TMB → TCAP กลายเป็นผู้ถือหุ้นใหญ่ของ TMB (ภายหลังเปลี่ยนชื่อ TTB) อีกชั้น ทำให้ TCAP ถือ asset value มหาศาล แต่ market cap สะท้อนน้อย (paraphrase from sector reporting; primary verbatim quote ของ ดร.นิเวศน์ ที่อธิบายตรง ๆ ผู้รวบรวมยังไม่พบ).

**Hidden value pattern เดียวกับ QH** — โดย QH ถือ HMPRO 19.87% > QH market cap → ดู [08-case-qh.md](08-case-qh.md).

## Lessons (สำหรับ MaxMahon framework)

1. **Mid-tier เก่ง > big-cap อิ่มตัว** — เลี่ยง KBANK/SCB/BBL ไปหา TCAP ได้ runway มากกว่า
2. **PE/PBV/Yield 3 ชั้น = safety net เต็มสูตร**
3. **Hidden asset bonus** — TCAP มี TMB/TTB stake เป็น optionality ในงบ
4. **Restructure as catalyst** — การควบรวมธนชาต-TMB เป็น value unlock event ที่ ดร.นิเวศน์ ดักไว้แต่แรก
5. **ปันผลสม่ำเสมอ ≥ 3 บาท/หุ้น 3 ปี = pass 5-5-5-5 second 5** (yield บน raw cost ~10%)
