Save / Discard buttons — admin only · ซ่อนจาก viewer
Read-only คืออะไร
  • ค่าทุกตัว เห็นเหมือนกัน — slider position, day chip active, radio card active, switch on/off
  • แต่ กดอะไรไม่ได้ — slider drag ไม่ได้, day chip กดไม่ได้, radio card กดไม่ได้, switch toggle ไม่ได้

  • Backend ก็ต้อง check role ด้วย — POST /api/settings + POST /api/admin/* return 403 ถ้าไม่ใช่ admin

SCENE 06

ข้อมูลแยกอะไร / รวมอะไร

— สรุปว่า user 2 คนใช้อะไรร่วมกัน vs แยกกัน
▼ แยกต่อ user
  • Watchlist — หุ้นที่ติดตาม
  • Blacklist — หุ้นที่ไม่เอา
  • Notes — บันทึกต่อหุ้น
  • Custom lists — list ที่ตั้งเอง
  • Portfolio — generate จาก watchlist ตัวเอง (auto-แยก)
▼ ใช้ร่วมกัน (admin คุม)
  • Settings — schedule, Niwes filters, universe
  • Screener results — ผลสแกน 933 หุ้น (รันสัปดาห์ละครั้ง)
  • Stock fundamentals — ข้อมูลพื้นฐานหุ้น (SETSMART/thaifin)
  • Price cache — ราคาล่าสุด
  • History — scan ย้อนหลัง
หลักคิด — Settings + screener = "house rules" ของร้าน admin คุม / Watchlist + portfolio = "ตะกร้าของฉัน" user แต่ละคนแยก. เมียจะเห็น screener ผลเดียวกับมึงเสมอ แต่หยิบใส่ตะกร้าตัวเองได้
